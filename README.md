# Note2be
Welcome to the Note2be project! This project is a note-taking web application inspired by the popular Evernote app. It allows users to create, edit, delete, and organize notes seamlessly with a focus on simplicity and efficiency. Live at https://capstone-project-xsl6.onrender.com/

<img width="1346" alt="Screenshot 2024-08-18 at 12 24 59 PM" src="https://github.com/user-attachments/assets/a6067908-6d5d-49d6-9cbf-e33a3cd41022">

## Features
- User Authentication: Secure login and registration functionality using JWT (JSON Web Tokens) for session management.
- Create Notes: Add new notes with a title and content.
- Edit Notes: Modify the existing notes in real time.
- Delete Notes: Remove unwanted notes with ease.
- Rich Text Editor: Supports formatting for text (bold, italics, lists, etc.) for more expressive notes.
- Note Organization: Group notes into notebooks or categories for better structure.
- Responsive Design: Fully responsive UI, accessible from both mobile and desktop.

## Usage
- Sign up for an account or log in if you already have one.
- Create a new notebook and add notes to it.
- Format your notes using the rich text editor.

## Technologies Used
### Frontend
- React - UI framework
- Redux - State management
- React Router - For client-side routing
- Vite - Frontend build tool
- js-cookie - For handling cookies
- Axios - For API requests

<img width="1396" alt="Scr eenshot 2024-08-18 at 12 28 06 PM" src="https://github.com/user-attachments/assets/a7e151db-d51b-42a8-bca1-6167f6bdb47c">

### Backend
- Flask - Web framework
- SQLAlchemy - ORM for database management
- Alembic - Database migrations
- Flask-WTF - Form handling
- Flask-CORS - Cross-Origin Resource Sharing
- Flask-Login - User session management
- Gunicorn - Production WSGI server
- Werkzeug - WSGI utility library
- SQLite (Development) / PostgreSQL (Production) - Database
